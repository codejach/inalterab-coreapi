## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

List things you need to use the API and how to install them.

* docker
  ```sh
  docker ^20.10 
  ```
* npm
  ```sh
  npm install
  ```

### Installation

1. Execute the next command
   ```sh
   make setup
   ```
   ```sh
   make install
   ```
2. Enter environment var in `.env`
   ```js
   ENVIRONMENT=dev
   ```
